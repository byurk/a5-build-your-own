# Task

# Tool Usage

# Citations

Example: "When sharing information from documents, cite the source (e.g., 'According to lincoln_biography.txt...'). Include specific quotes when they add value."

# Format

REPLACE_ME - How should responses be structured?

Example: "Structure responses clearly:
